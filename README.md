# remember
# 🚀 Code Predictor 

## 📌 项目介绍

**Code Predictor  是一个功能完整、功能强大的在线代码编辑器，专为学习和开发设计。

### ✨ 核心特色

| 特性 | 说明 |
|------|------|
| 💻 **三语言编辑** | 同时支持 HTML、CSS、JavaScript 编辑 |
| 👁️ **实时预览** | 代码实时渲染到预览窗口 |
| 💡 **AI 代码预测** | 智能预测下一步代码建议 |
| 📚 **代码索引解释** | 100+ 标签、属性、方法的完整解释 |
| 🔄 **反向映射系统** | 预览中的改变自动同步到代码 |
| 🔍 **元素检查器** | 点击预览元素查看和编辑属性 |
| 📋 **预设模板** | 3 个内置模板快速开始 |
| 💾 **本地保存** | 代码自动保存到浏览器存储 |

## 📞 常见问题

### Q: 代码会丢失吗？
**A:** 不会。每次点击 💾 保存会存到 localStorage，刷新也不丢失。

### Q: 如何学习代码含义？
**A:** 点击 📚 代码索引查看详细解释。

### Q: 如何修改预览中的元素？
**A:** 点击预览中的元素打开检查器面板。

### Q: 支持多少种语言？
**A:** 支持 HTML、CSS、JavaScript 三种。

### Q: 预览可以交互吗？
**A:** 可以。表单、按钮等元素可以交互，JavaScript 会运行。
